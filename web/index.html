The contract

HereNow is intentionally small. It is not ambient. It is not persistent. It does nothing unless you ask.

Idle by default
This page never activates chat. No prompts. No networking.
Explicit opt-in per session
Opening the chat page still does nothing until you consent for that tab.
Close tab = chat off
Participation ends immediately when the chat tab closes. No memory retained.
No discovery
No global rooms, no public directory, no “who’s online”, no algorithmic surfaces.

What it is (and isn’t)

  • It is a short-lived chat session between people who have a link.
  • It is consent-first: nothing happens without an explicit “yes”.
  • It is not a social network, community platform, or identity system.
  • It is not a background service. If you’re not on the chat page, it’s asleep.

How it works (high level)

A chat session is a temporary “room” created by your browser. People join only by opening the chat link and consenting.

  • Messages flow peer-to-peer using WebRTC.
  • A Cloudflare Worker + Durable Object is used only for signaling (introductions).
  • No accounts. No saved history. No replay. No recovery.

Privacy posture

  • No analytics on this landing page.
  • No scripts on this landing page.
  • No background polling. No “phone home”.
  • Chat is ephemeral by design; closing the tab ends the session.

Use it safely

This is a lightweight tool, not a moderation platform. Keep it boring:

  • Share room links only with people you intend to talk to.
  • Assume participants can screenshot.
  • If you don’t trust the room, don’t join it.

FAQ

Does visiting this page start anything?
No. This page is inert by design.

Does opening the chat page auto-join?
No. The chat page will ask for explicit consent per session.

Is there a “home room” or public lobby?
No. No global rooms, no discovery.

Is anything stored?
No chat history, no accounts, no persistence. The only “state” is what exists while your chat tab is open.
