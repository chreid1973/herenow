Open chat

The simple rules

HereNow is intentionally small. It’s not a platform. It’s a moment. If you don’t open the chat page and say yes, nothing happens.

How to start

The chat page is a two-step flow: set your room rules first, then turn chat on.

  • Step 1 — Room setup: choose a display name, and optionally enable Knock mode (passphrase).
  • Step 2 — Turn chat on: a share link is created. Copy it and send it to the people you want.

If Knock mode is enabled, the link includes a tiny “knock” flag. Joiners must enter the passphrase first, then they can turn chat on. The passphrase is never included in the link.

What it is (and isn’t)

  • It is a short-lived chat session between people who have the link.
  • It is consent-first: it stays off until you explicitly turn it on.
  • It is not a social network, community platform, or identity system.
  • It is not a background service. If you’re not on the chat page, it’s asleep.

How it works (plain English)

Your browser creates a temporary chat link. People join only by opening that link and choosing to participate.

  • Once connected, messages go directly between browsers.
  • A tiny relay is used only for introductions (so peers can find each other).
  • No accounts. No saved history. No replay. No recovery.

Abuse brakes (built-in)

HereNow isn’t a platform, so the safety approach is simple: friction before participation, and quick exits.

  • Kick + 60-minute deny (this tab only) from the People drawer.
  • New link (room reset) to boot everyone and invalidate the old link.
  • Silent limits (message size + rate limits) to keep spam from melting the room.

Optional: Knock mode

If you want extra friction, you can require a passphrase. The link lets people connect, but they can’t chat until they enter it. The passphrase is never included in the link and nothing is saved.

Privacy posture

  • No analytics on this page.
  • No scripts on this page.
  • No background polling. No “phone home”.
  • Chat exists only while the chat tab is open.

Use it safely

This is a lightweight tool, not a moderation platform:

  • Share chat links only with people you intend to talk to.
  • Assume participants can screenshot.
  • If you don’t trust the link, don’t join.

FAQ

Does visiting this page start anything?
No. This page is inert by design.

Does opening the chat page auto-join?
No. You’ll set your room rules first, then you choose whether to turn chat on.

Is there a public lobby?
No. Only links you share exist.

What if the link says “Knock”?
That room requires a passphrase. Enter it first, then you can turn chat on.

Is anything stored?
No chat history, no accounts, no persistence.
